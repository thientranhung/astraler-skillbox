# Host Topology Fixtures

Templates and recipes for T1 Skill Host Folder topology cases. Copy this folder
into a run folder before mutating.

- `empty-host/`: valid empty host with `.agents/skills`.
- `malformed-md/`: host skills with missing and malformed `SKILL.md` variants.
- `duplicate-names/`: two skill folders with the same frontmatter name.
- `host-is-file.txt`: file used as a rejected host path.
